# Example Template

A minimal built-in template for trying out generator-kickstart.

```ejs package.json
{
  "name": "my-project",
  "version": "1.0.0",
  "description": "",
  "scripts": {
    "start": "node src/index.js"
  }
}
```

```ejs src/index.js
console.log('Hello from generator-kickstart!');
```

```ejs README.md
# My Project
```
